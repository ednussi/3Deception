Questionnaire

We designed and implemented a CIT-like subject-specific questionnaire, which is used in an interview with the subject (about 20-25 minutes) to acquire muscle movement timeseries using PrimeSense Carmine depth camera

Feature extraction

Our feature set includes 4 first moments of the muscle timeseries, dynamic muscle activation change rates, discrete activation levels, number of blinks and smiles, and more

Learning

We use Linear Support Vector Machine as our main classifier, though we also tested SGD, gradient boosting, single decision trees and random forest.

Data learning methods

Answer type prediction

Given a dataset of answer frames for 5 question types, train the classifier on 4 types using cross-validation.

Distinguish between truthful YES/NO

We also trained the classifier separately on only YES or NO answers, and got an improvement in classification accuracy.

Predict session type

Lastly, we trained the classifier to predict the session from which the answer was sampled.
